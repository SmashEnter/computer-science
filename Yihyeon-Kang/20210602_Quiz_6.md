## **10.다양한 자료 구조와 검색 시간 (오답)**

```
1번. 시간 복잡도
2번. 공간 복잡도
3번. 메모리 주소 표기법
4번. 데이터의 양
```

정답: 3번. 메모리 주소 표기법

오답: 4번. 데이터의 양  
3번, 4번 중에 메모리 주소 표기법이라는 건 구체적으로 뭘 말하는 건지 모르겠어서 찝찝한 마음을 안고 데이터의 양을 고름.
